UTF8_sanitizer originally by whitequark on github

https://github.com/whitequark/rack-utf8_sanitizer
edited by cpeppis july 2014
also based on utf8-cleaner by singlebrook on github
https://github.com/singlebrook/utf8-cleaner
also uses some code/ideas from http://stackoverflow.com/a/8873922

Middleware to sanitize input to the server in order to avoid problems with invalid encoding from users. This has been a big problem with a number of spiders that appear to ignore the robots.txt (mainly one from a Chinese search engine called Easou). They make random requests to the server with POST data full of random crap that used to cause several 500 error warning emails a day.

encoding: UTF-8
